Backbeat

  • Description: Given a .wav file and a target tempo, determines which measures were played too fast and which were played too slow.

  • Group project for CS 196.

  • Used the Essentia audio analysis library to determine whether the tempo of a .wav file was slower than the desired tempo.

  • Role: Installed and figured out how to work with the Essentia library. Wrote modularized functions to interface between the general algorithm and the Essentia Library.
